Physician Vacation Manger User Manual

Overview

This system provides a User Interface with which physicians can enter planned vacations or general periods of time away.

The landing page for the system shows the calendar showing the physician currently scheduled time-away.

Below the calendar is a widget use to enter a time-away.

A typical view of the user interface is shown below:

Access to the System

The Time-away manager is reached by clicking on the Sa, Staff Availability, button on the ION landing page.

Physicists, Dosimetrists, and Physicians each have their own Time-Away manager and will appear on calendars including only their own group

Use by Physician Administrative Staff.

An authorized user can select a physician from the drop-down list, and create or edit a Time-away for this physician.

Entering a Vacation or Time-Away

To enter a vacation the user clicks on the Calendar Icon in the 'Enter a date range' UI element, and selects the dates for her Time Away

To enter a Time Away of single day duration, enter the SAME date twice.

The User then select a Reason and a Coverer from the Drop Down menus.

The User can select 'TBD' for the Coverer so that she can nominate a coverer later.

The Note field use is optional

WTM Coverage

If the User needs a change to her WTM date that check box is clicked.

2 buttons are then shown with which the user can either attest that she in covering the WTM herself or nominate another physcian to cover. As shown below.

If the User selects 'Self' a Calendar widget appears so that the User can select the changed WTM date.

If the User selects the 'Covering MD' then the Physician nominated in the 'Coverer' widget, will be notified that a new WTM date must be scheduled.

Color Codes

In the above example the colors indicate

  • Orange Background ---- Coverage has not been accepted.
  • Orange Background --- Coverage not accepted

Accepting Coverage

When a time-away is entered the physician nominated for coverage receives an email with a link which can be uses to accept of decline the coverage.

If the Coverer is not expected to cover WTM, a simple Accept/Decline choice is presented

If the Coverer is expected to cover WTM, the acceptance User Interface includes a widget to choose a WTM date.

Changing or Deleting a Time Away

If a user needs to change or delete her Time Away, she can click on the Time Away to see a User interface Element which can be used to change the parameters of the Time Away
